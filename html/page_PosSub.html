PosSub() is a function performing positive subtraction:

PosSub(a,b) = a-b if( a>=b ) and 0 otherwise. 


template <class UInt,class ... TT>
PosSubResult<UInt> PosSub(UInt value,TT ... tt);

PosSub() accepts any number of arguments, value type is determined from the first one, it must be an unsigned integral type, other arguments must be converted to this type. Extra arguments are subtracted one by one from the first one.

The resulting type encoded both result and underflow flag, it can be implicitly converted to result. 

template <class UInt>
struct PosSubResult
 {
  UInt value;
  bool ok;
  
  operator UInt() const { return value; }

 };
