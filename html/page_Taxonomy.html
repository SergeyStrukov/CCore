Taxonomy of classes

This page provides a list of common terms and the taxonomy of classes. Class in C++ is a very generic and powerfull entity, so we classify them by its purpose and nature.

Terms

Efficient means O(1)-nothrow. I.e. it is a sequence of operations, which has a definite reasonably small execution time limit and has no error outcome.

Small Data Type is a type like basic types with a small state and data semantic. These types can be defined as types with the efficient copy constructor and assign operator.

Derivability is an ability of some entity to be used as a building block of the whole. See, for example, Swap.

Classification

Word is a enumerator, purposed to verbose some generic action. In C++ there is no named constructors. Sometimes you have to used unused arguments of Word type to clearly distinguish between different overloaded constructors. See, for example, Nothing.
