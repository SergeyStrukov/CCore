Replace

Replace...() is a family of functions, they replace the first argument with a new value and return the old value.

The basic of them is a simple Replace(). 


template <class T,class S>
T Replace(T &t,S a); // t <- T(a)

There is also a set of variations: 


template <class T>
T Replace_null(T &t); // t <- T()

template <class T>
T Replace_min(T &t,T a); // t <- Min(t,a)
 
template <class T>
T Replace_max(T &t,T a); // t <- Max(t,a)
 
template <class T,class S>
T Replace_add(T &t,S a); // t <- t+a
 
template <class T,class S>
T Replace_sub(T &t,S a); // t <- t-a

The final one is a generic, you can use it with lambdas: 


template 
T Replace_gen(Func func,T &t); // t <- func(t)
