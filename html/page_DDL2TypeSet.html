DDL2TypeSet

DDL2TypeSet is a tool for generation a TypeSet class. This class can be used to map DDL2 data using the DDL2::TypedMap. 


DDL2TypeSet.exe <ddl-file-name> <typdef-file> <typeset-file>

ddl-file-name is the name of a DDL2 file, which contains type definitions.

typedef-file will contain TypeDef declarations.

typeset-file will contain TypeSet declarations.

For example, see files LangTypes.ddl, LangTypeDef.gen.h, LangTypeSet.gen.h in the directory CCORE_ROOT/tools/CondLangLR1.
