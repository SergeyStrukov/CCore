DefaultValue

Files CCore/inc/gadget/DefaultValue.h CCore/src/gadget/DefaultValue.cpp

DefaultValue() can be used to create an object with default initialization (i.e. using default constructor): 


C obj=DefaultValue();

Or:


void func(C arg=DefaultValue());

The type C above must be copyable.

DefaultInitObject can be used to create an object with default initialization without copying: 

DefaultInitObject<C> init; // init.obj of type C is default initialized

Both of DefaultValue() and DefaultInitObject are mostly to make code more verbose and sometimes more clear written.
