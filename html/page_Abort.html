Abort

Files CCore/inc/Abort.h CCore/src/AbortMsgFunc.cpp HCore/src/Abort.cpp XCore/src/Abort.cpp

There are two functons to abort an execution with a diagnostic message: 

void Abort(StrLen str);
 
void Abort(const char *zstr);

One of them accept a message text as a StrLen argument, another one as a zero-terminated character string. An exact method of the sending diagnostic message is target-dependent and implemented differently for HCore and XCore targets.

There is also the global constant function pointer, which can be used to alter the Abort() behavior. 

/* types */ 

typedef void (*AbortMsgFuncType)(StrLen str);

/* global const AbortMsgFunc */ 

extern const AbortMsgFuncType AbortMsgFunc;

You may link an object file in your project with this constant defined to introduce a hook in Abort().
