EchoDevice

Files CCore/inc/net/EchoDevice.h CCore/src/net/EchoDevice.cpp

The class EchoDevice implements echo service. It can be attached to a packet multi-point device. Been attached the service simply reflects incoming packets. This class is useful in net testing. 


class EchoDevice : NoCopy
 {
   ....

  public:
  
   enum Event
    {
     Event_inbound,
     
     Event_nopacket,
     Event_badformat,
    
     EventLim
    };
    
   friend const char * GetTextDesc(Event ev); 
   
   typedef Counters<Event,EventLim> StatInfo;
    
  public: 
  
   static const ulen DefaultMaxPackets = 200 ;
  
   explicit EchoDevice(StrLen mp_dev_name,ulen max_packets=DefaultMaxPackets);
   
   ~EchoDevice();
   
   void getStat(StatInfo &ret);
   
   uint64 getTraffic();
 };

Constructor attaches the object to the packet multi-point device registered under the given name. max_packets limits the packet usage.

getStat() returns device statistic.

getTraffic() returns device traffic counter.
