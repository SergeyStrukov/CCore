TickTimer is a helper class, it can be used to implement periodical "ticks". 


class TickTimer 
 {
   MSecTimer timer;
   unsigned period;
   
  public:
   
   explicit TickTimer(MSec period);
   
   bool poll();
   
   MSec remains() const; 
 };

The constructor takes the period as the argument. The class uses the internal millisecond timer to count time.

poll() tests the elapsed time against the period. If it is less, then the method returns false. If the elapsed time is greater than the period, but less than the 2*period, it returns true and shifts the timer by the period. Otherwise, it resets the timer completely and returns true.

remains() returns the remaining time to the next tick, if the elapsed time if greater than the period, the return value is null.
