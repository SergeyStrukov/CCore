StrKey

Files CCore/inc/StrKey.h CCore/src/StrKey.cpp

StrKey is a simple class. It is a combination of StrLen class and hash code. This class can be used as the key type to build maps. It speeds-up string comparison. 


struct StrKey : CmpComparable<StrKey>
 {
  uint32 hash;
  StrLen str;
  
  StrKey();
  
  explicit StrKey(StrLen str);
  
  CmpResult objCmp(const StrKey &obj) const;
 };
