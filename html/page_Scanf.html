Scanning is a process, inverse to the printing. I.g. scanning is a creation of an object value from a sequence of characters. There are three actors in a scanning process: the object, the input device and scannig options. The input device provides a character sequence, scanning options determines how the object value is encoded. The type of the object is responsible for the scanning implementation. There are default implementations for integral and string types.
