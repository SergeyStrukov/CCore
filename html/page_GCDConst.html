GCDConst

Files CCore/inc/algon/GCDConst.h CCore/src/algon/GCDConst.cpp

Here is two algorithms to calculate GCD (great common divisor) and LCM (least common multiple) in the compile-time. Arguments are template parameters of some unsigned integral type. Result() is a constexpr function, which return the GCD or LCM of arguments respectively. 


template <class UInt,UInt A,UInt B>
struct Algon::GCDConst
 {
  static constexpr UInt Result(); // return GCD(A,B)
 };
 


template <class UInt,UInt A,UInt B>
struct Algon::LCMConst
 {
  static constexpr UInt Result(); // return LCM(A,B)
 };
